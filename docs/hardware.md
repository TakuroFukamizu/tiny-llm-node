# Hardware

## Host

Raspberry Pi 5

Initial configuration uses 8GB RAM model.

16GB may be used if needed.

---

## PCIe expansion

Geekworm X1010 PCIe expansion board.

Provides PCIe slot for GPU connection.

---

## GPU

Initial target:

RTX 4060 8GB

Requirements:

- low power
- compact size
- good tensor performance

---

## Cooling

Active cooling recommended for:

- GPU
- DC-DC converter
- Raspberry Pi 5

---

## Mechanical considerations

GPU support bracket recommended.

Avoid stress on PCIe connector.
